## TH

อัปเดตวันที่ 18 กุมภาพันธ์ 2026 เราเพิ่ม external tools และปรับโครง navigation เพื่อให้แยกประเภทเครื่องมือได้ชัดขึ้นตั้งแต่หน้ารวม

### สิ่งที่เปลี่ยน

- เพิ่มรายการเครื่องมือภายนอก `Word Counter` สำหรับงานเขียนและวัดจำนวนคำแบบรวดเร็ว
- เพิ่มรายการเครื่องมือภายนอก `Wheel Random` สำหรับสุ่มชื่อ งาน หรือกิจกรรม
- ปรับ dashboard และ sidebar ให้แยกกลุ่ม `internal` และ `external` ชัดเจน ลดความสับสนเวลาเลือกเครื่องมือ

## EN

On February 18, 2026, we expanded external tools and adjusted navigation structure to make tool categories clearer from the main entry points.

### What Changed

- Added `Word Counter` external entry for fast writing and word-count workflows.
- Added `Wheel Random` external entry for random name, task, and activity workflows.
- Updated dashboard and sidebar to clearly separate `internal` and `external` groups and reduce tool selection friction.
