## TH

อัปเดตวันที่ 20 กุมภาพันธ์ 2026 รอบนี้เป็นงานรีเฟรชโครงนำทางและหน้าใช้งานหลัก เพื่อให้ผู้ใช้เข้าถึงเครื่องมือและอัปเดตโปรดักต์ได้เร็วขึ้น

### สิ่งที่เปลี่ยน

- ปรับโครง dashboard ให้ลำดับส่วนชัดขึ้น และอ่านผลการค้นหาและการ์ดเครื่องมือได้ง่ายขึ้น
- ปรับ sidebar navigation พร้อมเพิ่มปุ่ม favorite ในรายการเครื่องมือโดยตรง
- ยกระดับ AI Assistant UX ด้วย status indicators, quick prompts และ composer behavior ที่ลื่นขึ้น
- เปิดหน้า Blog updates และย้ายทางเข้า Blog ไปที่ header navigation เพื่อเข้าถึงได้เร็วขึ้น
- ย้ายแหล่งข้อมูล blog ไปใช้ markdown (`src/content/blog/*.md`) พร้อมระบบ frontmatter parsing

## EN

This February 20, 2026 update delivered a navigation and core-page refresh to make tool access and product update discovery faster.

### What Changed

- Refined dashboard hierarchy with clearer search state and more readable tool cards.
- Improved sidebar navigation and added inline favorite toggles in tool items.
- Upgraded AI Assistant UX with better status indicators, quick prompts, and smoother composer behavior.
- Launched Blog updates page and moved Blog entry to header navigation for faster access.
- Switched blog data source to markdown (`src/content/blog/*.md`) with frontmatter parsing.
