## TH

อัปเดตวันที่ 19 กุมภาพันธ์ 2026 เราโฟกัสที่ flow การเริ่มต้นใช้งาน AI Assistant ให้สั้นลงและอ่านสถานะได้ชัดขึ้น

### สิ่งที่เปลี่ยน

- เพิ่มแถบสถานะผู้ช่วยเพื่อบอกสถานะ API key และการใช้ context ในรอบสนทนา
- ปรับ empty state ให้เริ่มแชตได้เร็วขึ้นด้วย quick-start prompts
- ปรับ hints และ submit behavior ใน composer ให้พิมพ์และส่งข้อความได้ลื่นไหลขึ้น

## EN

On February 19, 2026, we focused on reducing onboarding friction in AI Assistant and making chat status easier to understand.

### What Changed

- Added assistant status strip to surface API key state and context usage during chat.
- Improved empty state with quick-start prompts for faster first interaction.
- Refined composer hints and submit behavior for a smoother typing and sending flow.
